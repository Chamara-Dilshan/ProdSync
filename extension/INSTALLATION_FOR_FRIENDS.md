# ProdSync Extension - Installation Guide

## 📦 What You Need

- Google Chrome browser
- The ProdSync extension ZIP file

## 🚀 Installation Steps

### Step 1: Extract the ZIP

1. Locate the `ProdSync-Extension.zip` file
2. Right-click → "Extract All..."
3. Choose a location (e.g., Desktop or Documents)
4. Remember this location!

### Step 2: Enable Developer Mode in Chrome

1. Open Google Chrome
2. Type `chrome://extensions/` in the address bar and press Enter
3. Find the **"Developer mode"** toggle in the top-right corner
4. Turn it **ON** (it should turn blue)

### Step 3: Load the Extension

1. You'll see three new buttons appear: "Load unpacked", "Pack extension", "Update"
2. Click **"Load unpacked"**
3. Navigate to the extracted folder from Step 1
4. Select the **`dist`** folder (NOT the parent folder)
5. Click "Select Folder"

### Step 4: Verify Installation

✅ You should now see "ProdSync" in your extensions list!

- The extension icon should appear in your Chrome toolbar (top-right, puzzle piece icon)
- If you don't see it, click the puzzle piece icon and pin ProdSync

## 🎯 How to Use

1. **First Time Setup:**
   - Click the ProdSync extension icon
   - Sign in with your account (same credentials as the web app)

2. **On Etsy:**
   - Go to https://www.etsy.com/messages
   - You should see a "✨ Generate Reply with ProdSync" button near the message textarea
   - Click it to generate AI-powered replies!

## ⚠️ Important Notes

- **"Developer mode" warning:** You'll see a banner saying "Extensions in developer mode". This is normal for unpacked extensions - you can ignore it.

- **Updates:** When I send you a new version:
  1. Delete the old extracted folder
  2. Extract the new ZIP
  3. Go to `chrome://extensions/`
  4. Click the refresh icon on the ProdSync extension card

- **Troubleshooting:**
  - If the button doesn't appear on Etsy, reload the page
  - If login fails, make sure you're using the correct credentials from the main ProdSync app
  - Check the browser console (F12) for any error messages

## 🆘 Need Help?

If something doesn't work:

1. Open Chrome DevTools (press F12)
2. Go to the Console tab
3. Look for messages starting with `[ProdSync]`
4. Send me a screenshot of any errors

## 🔒 Privacy & Security

- Your API keys are encrypted and stored securely
- No data is shared with third parties
